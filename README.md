## 🌟 Featured Project: Aurora International Clinic Agent

I developed an end-to-end **Doctor Appointment Booking Agent** that moves beyond simple chatbots into a fully functional service.

- **State Management:** Uses **LangGraph** to manage complex conversational states without fragile regex.
- **Automated Notifications:** Integrated with **SMTP** for real-time email confirmations upon booking.
- **Persistence:** Structured data storage using **SQLite** to manage patient records and doctor availability.
- **Production-Ready:** Follows modular design principles, making it easy to scale and maintain.
